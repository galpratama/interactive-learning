Tentang Modul Tutorial Ini

Jika Anda belum pernah membangun sebuah situs web sebelumnya dan Anda tidak memiliki pengalaman coding atau desain, Modul Tutorial Web Design #1 - 7 Hari Jago Web Design (Membuat Website Portofolio) ini adalah materi yang cocok untuk anda. Dalam proyek ini, kita belajar bagaimana membangun sebuah website portofolio modern untuk desktop, tablet, dan perangkat mobile. Kita mulai dengan HTML dasar dan sintaks CSS. Berikutnya, kita belajar bagaimana membangun halaman web kustom dengan galeri foto dan halaman kontak. Selanjutnya, kita akan belajar bagaimana caranya agar sebuah website bisa diakses langsung di web.

Yang akan kita pelajari

  • Cara menulis kode dasar HTML
  • Mendesain halaman web menggunakan CSS
  • Cara membeli domain dan hosting
  • Bagaimana cara meng-upload file

Informasi Tambahan

Apabila semua materi di dalam modul tutorial ini sudah anda pelajari semua, diharapkan anda melanjutkan ke materi selanjutnya, yaitu Modul Tutorial Web Design #2 - Jagoan CSS (Tutorial Dasar Mendalam CSS). Pada materi tersebut akan membahas secara dalam mengenai Cascading Style Sheets atau yang biasa kita sebut dengan CSS.
